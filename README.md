# Калькулятор стоимости вёрстки сайтов

Профессиональный интерактивный калькулятор для расчёта стоимости разработки сайтов.  
Сделан полностью с нуля на **Vanilla JavaScript** без библиотек.

## Особенности
- Полностью адаптивная вёрстка (семантическая разметка, `fieldset`, `legend`, `label for`)
- Расчёт в реальном времени
- Поддержка радиокнопок, чекбоксов, `<select>` и `<input type="range">`
- Универсальная формула для ползунка (работает при любом `step`)
- Безопасный код с optional chaining (`?.`)
- Красивое форматирование цены (1 234 567 ₽)
- Современный дизайн 2025 года (градиенты, тени, кастомный ползунок)

## Технологии
- HTML5
- CSS3 (Flexbox, кастомные стили ползунка)
- Vanilla JavaScript (ES6+)

## Как запустить
1. Склонировать репозиторий
2. Открыть `index.html` в браузере
